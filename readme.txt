1. Открыть консоль Phpmyadmin.
2. Скопировать содержимое файла sql.sql в соответствующее поле консоли.
3. Запустить эмулятор и открыть сайт по адресу http://localhost/tic_tac_toe/field.html ,
	предварительно, разместив папку "tic_tac_toe" в нужный каталог.
4. Логин и пароль к базе данных можно изменить в фале lib/connection.php
   По умолчанию стоит: Пользователь "root", Пароль "password".
